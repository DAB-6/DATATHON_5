# **1. 개요**

### **핵심결론 요약**

Gcar는 가격은 강점이지만 **고객센터·앱·반납·패널티로 이어지는 연쇄적 불편 때문에 평점이 낮다**.
이 구조적 문제를 개선하기 위해 `고객센터 UX·UI개편`과 `만차 사전 안내 기능`을 제안하며, **평점 개선·부정 리뷰 감소·기존 고객 락인 강화를 목표**로 한다.

---

# **2. 분석 배경 및 목적**

### **문제 정의**

1. **브랜드 포지션의 모호함**
    - Gcar는 선발주자였으나 현재 시장 주도권은 Socar가 보유하고 있음.
    - 리브랜딩 이후에도 **뚜렷한 강점이 체감되지 않는다는 평판**이 다수 존재.
    - 가격 강점이 있으나, 가성비 브랜드로서의 명확한 포지션 형성에 실패.
2. **차고지 관련 초기 가설 검토**
    - 여러 기사와 보도자료에서는 Gcar가 차고지 수·운영 규모 면에서 경쟁사 대비 열세라는 지적이 반복적으로 언급됨.
    - 이에 따라 초기에는 “평점 하락이 차고지 부족·입지 문제 때문일 가능성”을 고려함.
    - 그러나 실제 리뷰 텍스트에서는 차고지·존 관련 불만이 적은 비중을 차지하는 것이 확인됨.
    - 따라서 차고지 문제는 고객 불만의 핵심 원인 아니라는 것으로 정리되었으며, 이후 분석은  앱 리뷰 분석을 통한 서비스 개선 전략에 집중함.
3. **가격 경쟁력과 실제 서비스 경험 간 괴리**
    - 긍정 리뷰에서 ‘저렴함·할인’ 등 가격 키워드는 가장 고빈도.
    - 그러나 평균 평점(3.49)과 1점 비율(31.4%)은 경쟁사 대비 유의하게 낮음.
4. **연쇄적 불만 구조 존재**
    - “앱 오류 → 반납 실패 → 시간 초과 → 페널티 → 고객센터 문의”
        
        라는 **패턴이 반복적으로 등장**하며 브랜드 신뢰를 크게 저하시킴.
        

### **분석 목적**

- Gcar가 **어디에서, 어떤 흐름으로** 불만을 야기하는지 정량적으로 규명
- 불만이 **감소하고 있는 영역 vs 전혀 개선되지 않는 영역**을 구분
- 리뷰 기반으로 **고객 이탈 방지·평점 개선**을 위한 실질적인 실행안 제안

# **4. 분석 방법론**

### **1차 가설: 왜 Gcar의 성과가 경쟁사 대비 낮게 보일까?**

- 앱 리뷰 데이터를 크롤링해 확인한 결과, **Gcar의 앱 평점이 경쟁사보다 유의미하게 낮음.**
    
    → 특히 **1점 리뷰 비율에서 경쟁사 대비 큰 차이**가 확인됨.
    
    → 시장 선도 기업 + 최근 리브랜딩까지 진행했음에도 성적이 부진한 이유 파악 필요
    
- 이에 따라 경쟁사와 **앱 리뷰 기반 비교 분석**을 수행:
    - **평점 비교 및 통계적 검증**(Mann-Whitney U 검정)
    - **긍정 리뷰 단어 분석**: Gcar는 "저렴한 대여료"가 강점
    - **외부 설문조사(카셰어링 선택 기준)**와의 조합 분석
- 설문조사 결과에 따르면 고객들은 **① 낮은 대여료(82.7%) → ② 인근 차고지(접근성)(59.9%)**를 가장 중요한 기준으로 인식함.
- 긍정 리뷰 분석 결과, Gcar는 설문조사 1위 기준인 **저렴한 대여료 측면에서는 경쟁사 대비 우위**
    
    → 그렇다면 **2순위인 ‘접근성(차고지 수)’에서 차이가 나는지 확인이 필요함**
    
    → 실제 기사/보도자료 기반 비교 결과, **차고지·운영 차량·MAU 모두 Gcar가 경쟁사 대비 크게 낮음**
    

### 2차 가설: 4가지 주요 불만 키워드 검토

- 1차 가설 결과, **가격·접근성만으로는 성과 부진을 설명하기 어려움**
    
    → 리뷰 텍스트에서 나타나는 **서비스 경험 관련 불만**에 주목
    
- 부정 키워드 분석 결과, 다음 **4개 키워드 군이 차고지보다 더 큰 불만을 유발**
    
    → **고객센터 / 앱·어플 / 반납·예약 / 벌금·과금·페널티**
    
- 이에 따라, 각 키워드 군에 대해 **심층 리뷰 분석**을 수행:
    - 키워드별 **긍·부정 감정 비율** 비교
    - **브랜드별 별점 분포**(1점 비율 등) 비교
    - 주요 키워드(전화·시간·환불 등)와 맥락 확인
    - **2021–2025년 분기별 부정 비율 추이 + 추세 검정(OLS + Newey-West)** 진행
- 주요 분석 결과 요약
    - **고객센터**
        - 부정 비율 87% 수준, 1점 비중도 경쟁사보다 높음
        - 콜 연결 실패·불친절 응대에 대한 불만이 집중
        - 2021→2025년 사이 부정 비율은 **통계적으로 유의하게 감소(p<0.05)**
            
            → 개선 노력으로 불만은 완화되고 있으나, **여전히 부정이 긍정보다 많아 신뢰 회복은 미완**
            
    - **앱/어플**
        - 부정 비율 약 79%, 브랜드 간 별점 분포 차이는 크지 않음
        - 앱 기능 오류가 예약·반납 지연, 고객센터 문의로 이어지는 패턴
        - 2023년 리뉴얼 전후로 **일시적 안정화**는 있었으나, 장기적인 개선 추세로 보기는 어려움
            
            → **기술적 안정성·품질 관리 강화 필요**
            
    - **반납/예약**
        - 부정 비율 약 93%로 **가장 높은 문제 영역**, 1점 비중도 경쟁사 대비 크게 높음
        - 시간 지연, 예약 변경/취소, 환불 지연 등이 반복적으로 등장
        - 2023년 일시적 감소 이후 다시 기존 수준으로 회귀, **감소 추세는 통계적으로 유의하지 않음(p≈0.94)**
            
            → **반납/예약 프로세스 전반의 구조적 문제**가 여전히 해결되지 않음
            
    - **벌금/과금/페널티**
        - 부정 비율 약 91%, 1점 비율이 가장 높은 **고위험 키워드**
        - 시간 초과, 요금·페널티 기준 불명확, 환불 관련 불만이 주로 등장
        - 2021–2025년 동안 부정 비율은 **전반적으로 증가하는 양상**, 감소 추세는 전혀 관찰되지 않음
            
            → **요금/페널티 정책과 안내 방식 자체의 재설계가 필요**함
            
- 위 결과를 종합하면,
    - **앱 오류 → 예약·반납 지연 → 추가요금·페널티 부과 → 고객센터 문의 폭증**으로 이어지는 하나의 ‘문제 흐름’이 고객 여정 전반에서 반복되고 있음.
    - 따라서 Gcar의 개선 방향은 **개별 기능(앱/고객센터 등)을 따로 보는 것이 아니라, 고객 여정 전체를 고려한 통합 서비스 개선**에 두기로 함.
    : 주차장 혼잡도 사전 예측 및 안내, 고객센터·이용가이드 페이지 재설계 및 A/B 테스트

# **6. 결론 및 시사점**

### **핵심 인사이트 정리**

1. **문제점 — 고객 여정 전반에서 연쇄적으로 불만이 발생함**
    
    가격 경쟁력은 우위에 있으나, 앱 오류 → 반납·예약 지연 → 페널티 부과 → 고객센터 문의 증가로 이어지는 **연결된 불만 흐름**이 전체 만족도를 구조적으로 저하시킴.
    
2. **개선 방향 — 단일 기능이 아닌 ‘고객 여정 흐름’ 중심의 통합 개선 필요**
    
    혼잡도 예측, 앱 안정성 강화, 반납/예약 안내 고도화, 고객센터·가이드 재설계 등을 통해 문제 발생 지점을 사전에 차단하고, 단계 간 불편이 축적되지 않도록 개선해야 함.
    
3. **기대효과 — 불만 흐름 해소를 통한 평점 회복 및 CS 감소**
    
    주요 불만 원인의 연결 고리가 약화되면서, 반납 지연·과금·문의 등 핵심 이슈가 감소하고 결과적으로 **평점·충성도 회복과 연계된 긍정적 효과**가 기대됨.
